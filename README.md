# Hospitality Data Analysis

## Project Overview

This project analyzes hotel booking data using Python and Pandas.

The objective is to explore booking trends, identify data quality issues, analyze hotel capacity, and generate business insights using data visualization.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset

The project uses hospitality booking datasets including:

- fact_bookings
- fact_aggregated_bookings
- dim_hotels
- dim_rooms
- dim_date

## Project Tasks

- Data Import
- Data Exploration
- Data Cleaning
- Handling Missing Values
- Removing Invalid Records
- Outlier Detection
- Booking Analysis
- Capacity Analysis
- Revenue Analysis
- Data Visualization

## Key Insights

- Identified invalid guest records.
- Detected revenue outliers.
- Compared hotel categories.
- Analyzed booking platforms.
- Found properties with the highest capacity.
- Identified overbooked days.
